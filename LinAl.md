### 20. Линейное векторное пространство. Аксиомы линейного пространства. Примеры. Линейная независимость векторов.

  #### Линейное или векторное пространство
**Линейным (векторным)** пространством называется множество **V** произвольных элементов, называемых векторами, в котором определены операции *сложения векторов и умножения вектора на число*, т.е. любым двум векторам **x** и **y** поставлен в соответствие вектор **x + y**, называемый суммой векторов **x** и **y**, любому вектору **x** и любому числу **α** из поля действительных чисел **R** поставлен в соответствие вектор **α·x**, называемый произведением вектора **x** на число **α**

  #### Аксиомы линейного пространства
1. Каждой паре элементов ***x*** и ***y из L*** отвечает элемент ***x + y из L***, называемый суммой x и y, причём:
* *x + y = y + x* − сложение коммутативно;
* *x + (y + z) = (x + y) + z* − сложение ассоциативно;
* *x + 0 = x* − существует единственный нулевой элемент 0 ( *x + 0 = x для любого x из L*);
* *x + (− x) = 0* − для каждого элемента x из L существует единственный противоположный элемент −x ( *x + (−x) = 0 для любого x из L*).

2. Каждой паре ***x и α**, где ***α*** − число, а ***x элемент из L***, отвечает элемент ***α·x**, наываемый произведением α и x, причём:
* *α·(β·x) = (α·β)·x* − умножнение на число ассоциативно: ;
* *1·x = x* − для любого элемента x из L.

3. Операции ***сложения и умножения на число*** связаны соотношениями:
* *α·(x + y) = α·x + α·y* − умножнение на число дистрибутивно относительно сложения элементов;
* *(α + β)·x = α·x + β·x* − умножнение на вектор дистрибутивно относительно сложения чисел.

  #### Линейная независимость векторов
Если линейная комбинация $\lambda_1 \cdot \alpha^1 + \lambda_2 \cdot \alpha^2 + ... + \lambda_p \cdot \alpha^p$  ***может*** представлять собой *нулевой* вектор тогда, когда среди чисел $\lambda_1, \space \lambda_2 , \space ..., \space \lambda_p$ есть *хотя бы одно, отличное от нуля*, то система векторов $\alpha^1, \space \alpha^2, \space ..., \space \alpha^p$  называется ***линейно зависимой*** <br/>
Если линейная комбинация $\lambda_1 \cdot \alpha^1 + \lambda_2 \cdot \alpha^2 + ... + \lambda_p \cdot \alpha^p$ представляет собой нулевой вектор только тогда, когда *все числа* $\lambda_1 , \space \lambda_2 \space ,\space ..., \space \lambda_p$ *равны нулю*, то система векторов $\alpha^1, \space \alpha^2, \space ..., \space \alpha^p$ называется ***линейно независимой***

### 21. Базис линейного векторного пространства и координаты вектора. Разложение вектора линейного пространства по системе векторов. Размерность пространства.

  #### Базис линейного пространства
***Базисом*** линейного пространства называется *любая* упорядоченная максимальная линейно независимая системе его векторов. <br/>

*или* <br/>

***Базисом*** линейного пространства *L* называется любая упорядоченная система $a_1, \space а_2 \space , \space ..., \space а_n, \space ...$ (\*) его векторов, удовлетворяющая следующим требованиям:
* **любой** вектор из *L можно* представить в виде *линейной комбинации* конечного числа векторов из (\*);
* **ни один** вектор $a_к$ Из системы (\*) *нельзя* представить в виде *линейной комбинации* конечного числа остальных векторов из (\*).

*Упорядоченный набор коэффициентов*, с помощью которых данный вектор выражается через базисные векторы, называется **координатами** этого вектора в данном базисе
